[筆記] Install Python Gtk binding and Qt binding on SnowLeopard

其實也不用拿source自己編, 用MacPort就可了, 很簡單:

install PyQt4:

sudo port install py26-pyqt4

install Py-gtk

sudo port install py26-gtk
